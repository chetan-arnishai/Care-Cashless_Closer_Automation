How to Run / टूल कसे चालवायचे

  1. Enter the Claim / Intimation Number — no spaces.
    क्लेम नंबर टाका — स्पेस नसाव्यात.
  2. Enter the Patient Name exactly as in the case.
    पेशंटचे नाव केसमध्ये आहे तसे टाका.
  3. Click Browse PDF → Main Report (required).
    Main Report PDF निवडा — हे अनिवार्य आहे.
  4. Click Browse PDF → Doc_1 and Doc_2 (optional).
    Doc_1 आणि Doc_2 PDF निवडा.
  5. Click ▶ Start — PDF data will be extracted first, then automation runs.
    Enter OTP in the browser when prompted.
    Start दाबा — आधी PDF डेटा काढला जातो, मग automation सुरू होतो.
  6. Do NOT touch keyboard or mouse during automation.
    ऑटोमेशन सुरू असताना हात लावू नका.
📄 PDF Slots:
Main Report (file1) → Main case report — used for PDF extraction + fallback uploads
Doc_1 (file2) → Patient / Insured Part
Doc_2 (file3) → Hospital Part

जर Doc_1 किंवा Doc_2 नसेल तर Main Report वापरला जाईल.
NOTE: Upload new format PDFs only.
टीप: फक्त नवीन फॉरमॅट PDF अपलोड करा.
